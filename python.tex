\section{}
\begin{itemize}
\item hex()
\item raw 文字列
\item S.rjust(), S.ljust(), S.center() で幅を揃えられる
\item 数値のフォーマット ex. \%+010.2f
\item 辞書によるフォーマット ex. "\%(key)s" \% ({"key": "text"})
\item set 型の xor は A\textasciicircum B
\item 辞書では get() を使うとデフォルト値が指定できる
\item 空のリストや辞書は False
\item enumerate() や zip() をきちんと使おう
\item 関数の引数に * をつけるとタプルで、** をつけると辞書で受け取る
\end{itemize}
